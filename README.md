# Códigos Multipurpose


### Conectar a SQL Server
- Con este código podemos conectarnos a SQL server y extraer todos los datos de una o varias tablas y convertir los datosa  un dataframe para posteriormente poder trabajar con ellos.

### PDF Generator
- Con este código se crea un archivo PDF que contiene 4 tablas personalizables en su interior.

### Envío de correo electrónico
- Con este programa se puede enviar a uno o varios destinatarios un correo electrónico con uno o varois archivos adjuntos diferentes para cada destinatario.
